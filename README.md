# Catálogo de Filmes

Desafio técnico MyByte — SPA com Next.js.

## Rodar

```bash
npm i
npm run dev
```

http://localhost:3000

## Stack

Next 14 (App Router), TypeScript, Tailwind, shadcn.

## Funcionalidades

Listagem em cards, busca por título, favoritos (estado em memória), detalhe em `/filme/[id]`.

JSON em `data/filmes.json`.
